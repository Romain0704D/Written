### Markdown Support

The editor supports standard markdown syntax:

- Headers: `# H1`, `## H2`, `### H3`
- **Bold**: `**text**`
- *Italic*: `*text*`
- Lists: `- item` or `1. item`
- Links: `[text](url)`
- Code: `` `inline` `` or triple backticks for blocks
- Blockquotes: `> quote`
- And more!
